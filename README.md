## Instalação

Clone o repositório
 ```bash
 git clone git@github.com:Doe-IFBaiano/web.git
 ```
 
 Copie em um arquivo .env as variáveis necessárias e altere de acordo com o seu ambiente.
 ```bash
  cp .env.example .env

 ```
 Instale as dependências

 OP 1
 
 ```bash
 npm install

 ```

 OP 2

 ```bash
 yarn install
 ```
